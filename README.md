# Q-docs
Maana product documentation

## Build
Install mkDocs using [NPM](https://nodejs.org/en/):
```bash
npm i -g mkdocs
```

To produce a static site, within the root direcotry, simply:
```bash
mkdocs build
```
Hello

## Serve
To test the site:
```bash
mkdocs serve
```

## Deploy
To update GitHub's live view:
```bash
mkdocs gh-deploy
```
